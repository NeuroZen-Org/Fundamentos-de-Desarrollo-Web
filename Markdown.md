# Fundamentos de Desarrollo Web
## Resumen del Curso
Este curso de 1 hora introduce a estudiantes de secundaria a la creación de sitios web sencillos con HTML y CSS. **¡No
requiere descargas!** Solo abre tu navegador web.
**Duración total**: ~60 minutos
**Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación
**Prerrequisitos**: Ninguno
**Herramientas necesarias**: **¡Solo tu navegador web!** (Chrome, Firefox, Safari, Edge)
**� Repositorio de código fuente**: [https://github.com/NeuroZen-Org/Fundamentos-de-Desarrollo-Web.git](https://github.com/NeuroZen-Org/Fundamentos-de-Desarrollo-Web.git)
## Secuencia de la lección

### Lección 1: ¿Qué es un sitio web? + Estructura básica de HTML | Curso HTML y CSS #1
- **Descripción**: Veremos la estructura básica de HTML: etiquetas, elementos y atributos, para que entiendas cómo se construye cualquier página desde cero. ¡El inicio perfecto para tu camino en el desarrollo web!
- **Enlace**: [Ver la lección](https://youtu.be/nPGbvsnyHz4?si=FLBrCQkYKVAbKysM)
- **Empieza a programar**: [Open CodePen](https://codepen.io/pen/?template=your-starter)
  
### Lección 2: Encabezados, párrafos, listas, imágenes y enlaces + Primeros pasos con CSS | Curso HTML y CSS #2
- **Descripción**: Los elementos más comunes de HTML: encabezados, párrafos, listas, imágenes y enlaces. Además, te introducimos a CSS con selectores y propiedades esenciales como color, fuente y diseño. ¡Empieza a dar estilo a tus páginas!
- **Enlace**: [Ver la lección](https://youtu.be/uhV0bRiuJCA?si=gaqHKOqcg3MH5xTo)
- **Práctica**: [Clic para programar](https://codepen.io/pen/?template=your-starter) � **¡No necesitas registrarte!**
  
### Lección 3: Dale estilo a tu página: colores, bordes y contenido centrado + Tu primera web | Curso HTML y CSS #3
- **Descripción**:  Pondremos todo en práctica creando tu primera página web básica, como un perfil personal o una entrada de blog. ¡Tu primera web lista para mostrar!
- **Enlace**: [Ver la lección](https://youtu.be/9BXFQdTbwnk?si=3ly0MARGSOQJTJNn)
- **Práctica**: [Clic para programar](https://codepen.io/pen/?template=your-starter) � **¡Comienza al instante!**
  
### Lección 4: Errores comunes en HTML y CSS + Consejos prácticos para principiantes | Curso HTML y CSS #4
- **Descripción**: Los errores más frecuentes que cometen los principiantes en HTML y CSS. Te damos recomendaciones prácticas para escribir mejor código, evitar problemas y avanzar con confianza en tu aprendizaje. ¡Aprende de los errores y mejora rápido!
- **Enlace**: [Ver la lección](https://youtu.be/QSI0MPpuiWo?si=YqbHNOc-5ixz7FpA)
- **Práctica**: [Aplicar estilo con un clic](https://codepen.io/pen/?template=your-starter) � **¡No es necesario registrarse!**
