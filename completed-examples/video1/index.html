¿Qué es un sitio web?

Un sitio web es un conjunto de páginas que se pueden visitar desde un navegador. Sirve para compartir información, aprender, comunicarse y hasta comprar productos.

Ejemplos de sitios web

  • Google – buscador de información
  • YouTube – videos y entretenimiento
  • Wikipedia – enciclopedia en línea

En esta cartilla aprenderás cómo se construye un sitio web usando HTML y CSS.
